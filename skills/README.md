### Submit Your Skill

We welcome Skill contributions from everyone! To submit a new Skill:

1. Create your Skill JSON following the [Skill Development Guide](../docs/skill-development-guide.md)
2. Test it on a real device
3. Submit a Pull Request adding your JSON file to this directory
4. Use the [Skill Submission](https://github.com/zhaojiaqi/MeowHub/issues/new?template=skill_submission.md) issue template

## 技能库

本目录包含 MeowHub 的官方和社区贡献技能。每个技能是一个定义自动化任务的 JSON 文件。

| 技能 | 说明 |

| [朋友圈自动点赞](wechat-moments-auto-like.json) | 自动对朋友圈最新动态点赞 |
| [查看微信消息](wechat-check-messages.json) | 检查微信未读消息并汇报内容 |
| [添加微信好友](add-wechat-friend.json) | 通过微信号或手机号搜索并添加好友 |
| [短信摘要](sms-summary.json) | 查看最近短信并生成智能摘要 |

| 技能 | 说明 |

#### 日常（3 个）

| [闹钟快捷设置](set-alarm.json) | 快速设置闹钟 |

#### 购物（2 个）

### 提交你的技能

欢迎每个人贡献技能！提交步骤：

1. 按照 [Skill 开发指南](../docs/skill-development-guide.md) 创建技能 JSON
2. 在真机上测试
3. 提交 Pull Request，将你的 JSON 文件添加到本目录
4. 使用 [技能提交](https://github.com/zhaojiaqi/MeowHub/issues/new?template=skill_submission.md) Issue 模板
